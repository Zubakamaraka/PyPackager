- No-console `.exe` build

### How to Use
1. Run `py_packager.exe`
2. Drag in your `.py` file
3. (Optional) add `.ico` file
4. Choose output folder
5. Click **[ BUILD EXE ]**
6. Done! The `.exe` file will be created

### Requirements (if running from source)
- Python 3.x
- tkinter
- tkinterdnd2
- pyinstaller
